## tools list:

[printline](https://github.com/franklin-gaoxy/tools/blob/main/src/printline/readme.md)
> A CLI tool to generate and print separator lines or centered text in the terminal.

[simulate host usage tools](https://github.com/franklin-gaoxy/tools/tree/main/src/simulate%20host%20usage)
> A tool to simulate system resource usage, including CPU load, Memory consumption, and Disk I/O.

[Write Disk](https://github.com/franklin-gaoxy/tools/tree/main/src/write%20disk)
> A high-performance file generator tool designed to create large files or fill up disk space for testing purposes.

[localweb](https://github.com/franklin-gaoxy/tools/tree/main/src/localweb)
> A simple static file server that exposes a local directory over HTTP.

[many host command](https://github.com/franklin-gaoxy/tools/tree/main/src/many%20host%20command)
> A powerful SSH automation tool to execute commands or copy files across multiple hosts in parallel, serially, or in batches.

[host info](https://github.com/franklin-gaoxy/tools/tree/main/src/hostinfo)
> A lightweight host information collection server based on Gin framework (CPU, Memory, Disk, Network, etc).

[disk not sleep](https://github.com/franklin-gaoxy/tools/blob/main/src/disk_not_sleep/README.md)
> A utility tool designed to prevent hard drives (especially external HDDs) from entering sleep/spin-down mode.
